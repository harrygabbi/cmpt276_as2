About This Project

This project is a simple CRUD (Create, Read, Update, Delete) application to manage rectangles using the Java Spring framework. It was developed as part of an assignment for CMPT 276 at Simon Fraser University.

Features:

  • Add new rectangles to the database
  • Delete existing rectangles
  • Edit the details of existing rectangles
  • View information about all rectangles in the database

Technologies Used:

  • Java Spring Boot
  • Thymeleaf
  • Bootstrap
  • H2 Database

About the Developer:

This application was developed by Harry Gabbi, a computer science student at Simon Fraser University. This project is part of the coursework for CMPT 276, focusing on software development and engineering practices.
